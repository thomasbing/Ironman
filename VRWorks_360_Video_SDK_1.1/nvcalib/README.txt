This directory contains the camera calibration library and associated header files.

This folder contains the nvcalib API:

  nvcalib - Low Level Camera Calibration API
	header:   include/calibration_api.h
	lib file: library/nvcalib.lib
	.dll:     binary/nvcalib.dll
	For information on this API, refer to "VRWorks 360 Video Calibration Programming Guide" in this folder.

To experience the capabilities of this SDK, please refer to the sample app ..\samples\nvcalib_sample\run.bat

To build the nvcalib sample application, use CMake 3.2 or higher to generate a Visual Studio 2015/2017 x64 solution. The application can then be compiled from nvcalib_sample project.
