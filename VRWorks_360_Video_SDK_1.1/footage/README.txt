Readme footage
--------------

Folder contains sample footage files to be used for stitching. 
Use batch files in sample folders to run stitching with this footage.

Contents:
---------

Audio/Video/Images:
-------------------
Camera1.jpg .. Camera8.jpg - Sample JPEG files
Camera1.mp4 ... Camera8.mp4 - Sample MP4 files
input1, input2.pcm - Audio files

XMLs:
-----
image_input.xml - images input media feed XML
rig_spec.xml - Rig specification XML input for 
stitcher_spec.xml - Stitcher Specification XML
video_input.xml  - Video file input media feed XML
audio_rig_spec.xml - Audio Rig Specification XML
sample_calib_rig_spec.xml - Sample calibrated Rig Specification XML

PDFs:
-----
XML-Utility-Format-Specification.pdf: Refer to this guide for details on the XMLs Format
